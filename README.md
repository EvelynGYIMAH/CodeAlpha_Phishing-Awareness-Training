

# Cyber Security Task 2: Phishing Awareness Training
**Company:** CodeAlpha  
**Domain:** Cyber Security  

---

## Project Overview
This repository contains the training deliverables for **TASK 2: Phishing Awareness Training**. It features an educational, platform-independent markdown presentation mapping modern social engineering schemes, combined with an interactive risk assessment module.

### Included Training Components
* [Phishing Presentation Guide](Presentation.md) — Structural slides dissecting tactical variants, indicator signs, and real-world vector layouts.
* [Awareness Assessment Quiz](Quiz.md) — Interactive testing module evaluating structural vulnerability indicators.

## Core Training Objectives
1. **Identify Psychological Levers:** Understand how social engineers exploit urgency, authority, and fear to force rapid compliance.
2. **Technical Deconstruction:** Learn to spot spoofed sender domains, concealed hyper-links, and unsecure structural attachment formatting.
3. **Institutional Safeguards:** Establish out-of-band communication loops and MFA validation procedures to minimize threat exposure.
